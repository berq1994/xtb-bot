# Čeština ve výstupech

Tento balík převádí hlavní uživatelské výstupy více do češtiny:
- denní briefing
- Telegram náhled
- Telegram live status
- supervisor
- XTB ticket
- outcome tracking
- learning review
- performance review

Hlavní příkazy zůstávají stejné:
- `python run_agent.py production_cycle`
- `python run_agent.py telegram_live`
- `python run_agent.py daily_briefing`\n
