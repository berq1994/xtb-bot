Doporučený postup:
1. zkopíruj config/app_config.example.yml jako config/app_config.yml
2. python block14_config_check.py
3. python block14_telegram_test.py
4. python production_main.py

Povinné pro Telegram live:
- TELEGRAM_BOT_TOKEN
- TELEGRAM_CHAT_ID
- TELEGRAM_SEND_ENABLED=true

Výstupy:
- .state/block14_config_check.json
- .state/block14_telegram_test.json
- .state/block14_production_run.json
- production_report.txt
- logs/production.log
